A demo of basic mobile navigation. Any element can be used to trigger the panel that holds the menu. The panel is revealed via a fade animation, or can slide in from any direction.

Demo Usage:

$(function() {
  $('#panel').clearmenu({
    hasSubMenu: true,
    reveal: 'fade'
  });
});

Default options:

  {
  panel: this, // The element to act as the panel
  close: true, // Show a close button in the panel
  trigger: '.cm-hamburger', // The element that will trigger the selected panel when clicked
  reveal: 'fade', // The reveal animation. Can be fade, top, bottom, left, or right.
  hasSubMenu: true, // If the menu has should provide functionality for submenus or not
  subMenu: '.cm-submenu', // The class or ID of the submenu
  wordpress: false // If this panel is being used in a WordPress theme. This option will automatically use default WordPress menu class names for its functionality
  }
